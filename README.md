# Student Management System

A simple **console-based Student Management System** in Python that allows you to **add, view, search, update, and delete student records** stored in a CSV file.

---

## Features

- **Add Student** – Add a new student with Roll No, Name, Age, and Course.
- **View Students** – Display all student records in a tabular format.
- **Search Student** – Search for a student using Roll Number.
- **Update Student** – Update a student's details.
- **Delete Student** – Delete a student record by Roll Number.
- **Exit** – Exit the program safely.

---

## Technologies Used

- Python 3.x
- CSV module (for file handling)

---
